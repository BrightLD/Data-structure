#### 一看就懂的四种垂直居中

```
/*利用transfrom属性实现*/
 <style>
       html,body{
            width: 100%;
            height:100% ;
        }
       *{
           margin: 0;
           padding: 0;
       }
        #box2{
            position: absolute;
            width: 200px;
            height: 200px;
            top: 50%;
            left: 50%;
            background: yellow;
            transform: translate(-50%,-50%);
        }
```
```
/*flex/布局实现*/
 <style>
       html,body{
            
           display: flex;
           justify-content: center;
           align-items: center;
           width: 100%;
           height:100% ;
        }
       *{
           margin: 0;
           padding: 0;
       }
        #box2{
            width: 200px;
            height: 200px;
            background: yellow;
        }
    </style>
```
```
 <style>
        /*宽度和高度不知道的情况下*/
       html,body{
         
           width: 100%;
           height:100% ;
        }
       *{
           margin: 0;
           padding: 0;
       }
        #box2{
            width: 200px;
            height: 200px;
            position:absolute;
            left: 0;
            top: 0;
            right: 0;
            bottom: 0;
            margin: auto;
            background: yellow;
        }
    </style>
```
```
<style>
        /*固定宽度和高度的情况下*/
       html,body{

           width: 100%;
           height:100% ;
        }
       *{
           margin: 0;
           padding: 0;
       }
        #box2{
            position: fixed;
            width: 100px;
            height: 100px;
            top: 50%;
            left: 50%;
            margin-top: -50px;
            margin-left: -50px;
            /*margin-top和margin-left为盒子宽度和高度的负一般*/
            border:1px solid yellow;
            background :lightsalmon;
        }
    </style>

```
